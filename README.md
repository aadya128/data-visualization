# 🎞️ Letterboxd Movie Trends Dashboard

An interactive Tableau dashboard visualizing data from the Letterboxd dataset.  
It includes:

- ✅ Top-rated movies
- ✅ Genre distribution
- ✅ Yearly trends
- ✅ Clickable posters (with embedded web previews!)
